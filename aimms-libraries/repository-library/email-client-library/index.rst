Email Client Library
======================

.. meta:: 
   :description: The AIMMS Email Client Library allows users to send emails directly from AIMMS with support for templates, attachments, and SMTP server integration.
   :keywords: AIMMS Email Client Library, AIMMS email integration, Send emails from AIMMS, AIMMS SMTP email, AIMMS email API, AIMMS email templates, AIMMS email attachments, AIMMS HTML email, AIMMS automated emails, AIMMS email placeholders.

.. note:: Since AIMMS 24, the Email Client Library is no longer supported.  See also `AIMMS Product Lifecycle <https://documentation.aimms.com/deprecation-table.html>`_

The **Email Client Library** enables users to create and send emails directly from within AIMMS projects, 
supporting text and HTML templates with placeholders, attachments, and integration with SMTP servers for streamlined communication.
